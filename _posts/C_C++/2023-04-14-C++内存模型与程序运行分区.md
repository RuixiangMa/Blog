# 写在前面









|                    |         作用域         |       生命周期       | 内存区域 |
| :----------------: | :--------------------: | :------------------: | :------: |
|      全局内存      |        整个文件        |  应用程序的生命周期  |  全局区  |
|      静态内存      |    声明它的函数内部    |  应用程序的生命周期  |  全局区  |
| 自动内存(局部内存) |    声明它的函数内部    | 限制在函数执行时间内 |   栈区   |
|      动态内存      | 由引用该内存的指针决定 |     直到内存释放     |   堆区   |
